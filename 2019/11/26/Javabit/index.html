汉明距离
两个整数之间的汉明距离指的是这两个数字对应二进制位不同的位置的数目。
输入: x = 1, y = 4 输出: 2
解释:1 (0 0 0 1), 4 (0 1 0 0)
From Leetcode

用异或 a^b 解决：如果a、b两个值不相同，则异或结果为1。
1的个数即为汉明距离。
