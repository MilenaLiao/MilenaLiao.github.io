CSP Encoding:Non-binary to Binary

Author: Milena Liao Date: November 17, 2019  19:56:12 Category: Paper Notes

Reading notes about《Encodings of Non-Binary Constraint Satisfaction Problems》

  • Binary Constrant: A<B.
  • Non Constrant: A+B+C>10.

Preliminaries

Many CSPs can be formulated by non-binary relations.The methods of solving a non-binary CSP:

  1. Use algorithms like FC which have been generalized to non-binary constraints.
  2. Translate into a binary constraints.

Well-known methods for translating

Dual encoding

Hidden variable encoding

Hybrid encoding

Dual or Hidden

Double encoding

Both Dual and Hidden

Mapping between the encoding

  • Hidden variable encoding —(simplification)—> Dual encoding easily reverse!
  • In double encoding: Both original variables and dual variables, both constraints between original variables and dual variables.
  • Branching heuristics perform better on dual variable.

Theoretical comparison

In hidden variable encoding

  1. AC = GAC in original problem
  2. NIC = AC
  3. strong PC > SAC > AC

In dual encoding

  1. AC > GAC in original problem
  2. AC > AC on hidden

Conclusion

  • Show relationship between hidden variable and dual encoding.
  • Do more pruning in the dual.
  • These encoding translate non-binary CSP into a binary CSP. AC on hidden = GAC in original problem
  • Propose double encoding.
