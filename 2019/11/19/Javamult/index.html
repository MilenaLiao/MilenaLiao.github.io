Java数组乘积:时间复杂度限制

Author: Milena Liao Date: November 19, 2019  14:22:08 Category: Java

给定长度为 n 的整数数组 nums，其中 n>1，返回输出数组 output ，其中 output[i] 等于 nums 中除 nums[i] 之外其余各元素的乘积。
示例:
输入: [1,2,3,4]
输出: [24,12,8,6]
说明: 请不要使用除法，且在 O(n) 时间复杂度内完成此题
Leetcode

不占用额外空间：

O(n) 时间复杂度内，需要额外空间：
output[i] = nums[i]前所有数的乘积 * nums[i]后所有数的乘积
